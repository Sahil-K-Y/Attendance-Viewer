## 🎯 Overview

**Attendance Viewer** is a clean, web-based tool to track and visualize subject-wise attendance. Input your attendance data and instantly see your progress with beautiful visual indicators — no backend required!

🔗 **[Live Demo →](https://sahil-k-y.github.io/Attendance-Viewer/)**

---

## ✨ Features

| Feature | Description |
|:---|:---|
| 📊 **Subject-wise Lookup** | View attendance for each subject individually |
| 📈 **Progress Visualization** | Color-coded progress bars show attendance % |
| 🚦 **Status Indicators** | ✅ Safe / ⚠️ Warning / ❌ Danger zones |
| 📱 **Responsive Design** | Works on mobile, tablet & desktop |
| ⚡ **Instant Calculation** | No page reload needed — pure JavaScript |
| 🎨 **Clean UI** | Minimal, distraction-free interface |

---

## 🖥️ How It Works

```
Input your attendance data
         ↓
App calculates % per subject
         ↓
Visual progress bars rendered
         ↓
Status: Safe (75%+) / Warning (65-75%) / Danger (<65%)
```
